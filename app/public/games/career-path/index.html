How it works

  • The full club path is shown up front. No drip-feed — read it, think, type.
  • Three guesses per player. Each miss halves the points still on the table.
  • Typos are forgiven. There's no autocomplete to lean on, but the matching absorbs spelling slips — "Lewandowsky" still counts.
  • Close calls cost less. Almost-right answers get flagged as near misses instead of burning a full guess.
  • Difficulty mixes automatically. Every run blends easy, medium and hard careers — fact-checked, current transfers included.

Why no autocomplete?

Because autocomplete turns a recall game into a recognition game. Scrolling a dropdown until a name looks familiar isn't knowing football. On VerveQ you type the name you actually remember — the fuzzy matching handles the spelling, not the remembering.
